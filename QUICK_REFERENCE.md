# Billing Software Enhancements - Quick Reference

## Feature Overview

### 1. 📱 QR Code Upload  
**Location**: Below Payment Method section  
**How to Use**:
- Click file input to upload QR code image
- Check "Include in print" box to add to bills
- Preview shows before printing
- Click "Remove" to delete

### 2. **Bold** Text in Printed Bills
**Automatically Applied** - No user action needed  
**Bold Items**:
- 🏪 Shop Name
- 📄 GST Number  
- 👤 Customer Name
- #️⃣ Bill Number
- 💰 Total Amount, Payment Amount, Balance

### 3. ⌨️ Keyboard Navigation
**How to Use**:
1. Start in any "Item" field
2. Press **Enter** after each entry
3. Cursor auto-moves: Item → Weight → Rate → Next Item

**Benefit**: Faster data entry, less mouse usage

### 4. 🚶 Walk-in Customer Toggle
**Location**: Top of billing form (blue box)  
**How to Use**:
- ✅ Checked = Customer details required (default)
- ☐ Unchecked = Walk-in mode (no customer needed)

### 5. 📋 Phone Number Copy-Paste
**Already Working** - Use Ctrl+C / Ctrl+V normally

## Common Workflows

### Quick Walk-in Sale
1. Uncheck "Require customer details"
2. Fill in items using Enter key navigation
3. Enter payment amount
4. Click "Complete Bill"

### Sale with QR Payment Receipt
1. Upload your payment QR code (one-time setup)
2. Check "Include in print"
3. Complete bill normally
4. QR code prints automatically on receipt

### Fast Data Entry (Keyboard Only)
1. Tab to first "Item" field
2. Select item, press Enter
3. Type weight, press Enter  
4. Type rate, press Enter
5. Repeat for next item (auto-focus)

## Tips

- **QR Code**: Upload once, reuse for all bills
- **Keyboard Nav**: Works in all rows simultaneously
- **Walk-in Mode**: Great for quick cash sales
- **Bold Print**: Makes receipts more professional and easier to read

##Keyboard Shortcuts

| Key | Action |
|-----|--------|
| **Enter** (in billing table) | Move to next field |
| **Tab** | Move to next input |
| **Shift+Tab** | Move to previous input |
| Ctrl+V (phone field) | Paste copied number |
